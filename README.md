### I'm an early-career astrophysics and data-science researcher.

I hold an MSc in Physics (Astrophysics & Astroparticle Physics) from Friedrich-Alexander-Universität Erlangen-Nürnberg (FAU). My research focuses on multimessenger data analysis of high-energy astrophysical objects in the interstellar medium, such as supernova remnants, along with machine learning applications in astronomy and beyond. I mostly work on:

- High-energy astrophysics, multimessenger astronomy, astroparticle physics
  - Interstellar medium, supernova remnants, cosmic rays
- Data analysis, statistical methods, machine learning
  - Applied ML, generative AI (LLMs & VLMs)
